# Roadmap

## v0：CLI MVP

目标：

- LangGraph 工作流
- DeepSeek API 接入
- CLI 可运行
- Markdown 报告输出
- 本地 outputs 保存中间文件

## v1：增强研究质量

目标：

- 更好的搜索工具
- 网页正文抽取
- evidence 去重
- source ranking
- citation 管理
- critique fix rate 统计

## v2：FastAPI + 可视化

目标：

- FastAPI 后端
- 任务状态查询
- SSE 事件流
- 简单前端页面
- 展示 Plan、Sources、Evidence、Critique

## v3：多 Agent 并发研究

目标：

- Paper Research Agent
- GitHub Research Agent
- Blog Research Agent
- Official Docs Agent
- 多来源并发研究

## v4：本地知识库和 MCP

目标：

- 支持本地 PDF / Markdown
- 支持 RAG
- 支持 GitHub MCP
- 支持 Zotero / arXiv / 文件系统工具

## v5：招聘展示增强

目标：

- 增加项目架构图
- 增加完整 demo report
- 增加 README 截图
- 增加 benchmark case
- 增加简历项目描述和面试问答
