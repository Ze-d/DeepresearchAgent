Phase 3 设计：Human-in-the-Loop 审核节点

在自动化流程中插入人工把关，实现可控的 DeepResearch

1. 审核节点位置（推荐）

Plan --> research_agent × 4 --> Merge
                                     │
                                     ▼
                          ┌──────────────────────┐
                          │  🛑 Human Review      │
                          │  (interrupt before)   │
                          │                        │
                          │  审核内容:              │
                          │  • MergeSummary        │
                          │  • 所有 Sources        │

                                     │
                    ┌────────────────┼────────────────┐
                    ▼                ▼                ▼
               ✅ Approve     📝 Amend     🔄 Re-do
               继续 Critique    补充搜索/修改   回到 Research

2. 实现机制：LangGraph interrupt()

# nodes/human_review.py (新增)

from langgraph.types import interrupt

def make_human_review_node(llm):
    def human_review_node(state: AgentState) -> dict:
        """暂停执行，等待人工输入。"""
        merge_summary = state.get("merge_summary", {})

        # LangGraph interrupt: 暂停图执行，向外部暴露审核界面
        human_decision = interrupt({
            "action": "review_requested",
            "merge_summary": merge_summary,
            "sources": state.get("sources", []),
            "evidences": state.get("evidences", []),
            "conflicts": merge_summary.get("conflicts", []),
            "prompt": "请审核研究结果，选择 approve / amend / redo",
        })

        # human_decision 来自外部输入
        return {
            "human_review": human_decision,
            "status": f"reviewed_{human_decision['action']}",
        }

    return human_review_node

3. 三种审核动作

✅

Approve（批准）

审核通过，继续执行 Critique → Final。

{"action": "approve", "notes": "信息充分，可以继续"}
📝

Amend（补充）

提出额外搜索查询或修改建议，回 Research 阶段补充后再审核。

{"action": "amend", "new_queries": ["补充搜索...", "深入调研..."], "notes": "缺少性能对比数据"}
🔄

Redo（重做）

拒绝当前结果，修改研究方向后重新执行整个流程。

{"action": "redo", "revised_plan": {...}, "reason": "方向完全错误"}

4. API 暴露

# server/routes.py — 新增 endpoint

@router.post("/api/tasks/{task_id}/review")
def submit_review(task_id: str, decision: ReviewDecision):
    """
    提交人工审核结果，恢复被 interrupt 的图执行。

    Request Body:
      {"action": "approve" | "amend" | "redo",
       "notes": "...",
       "new_queries": [...],       // amend 时需要
       "revised_plan": {...}}      // redo 时需要
    """
    # 将 decision 注入到 graph 的 interrupt 恢复点
    graph_manager.resume(task_id, decision.dict())
    return {"status": "resumed"}
